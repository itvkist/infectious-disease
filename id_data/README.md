# Data Generation and Prediction API
 
## Quick start

```bash
$ # Get the code
$ git clone https://github.com/itvkist2/id_data.git
$ cd id_data
$ 
$ # Generate new data based on real dataset and save to sample_data.csv
$ python3 DataGeneration.py
$
$ # Start the server to predict future data
$ python3 Server.py
$
```
